Основные фактуры строительства

Блок рядовой

Основной строительный блок при возведении стен - при строительстве данных блоков используется преобладающее количество.

Блок рядовой половинчатый

Дополнительный строительный блок по отношению к рядовому строительному блоку

Блок угловой наружный.

Для формирования углов. Быстрое возведение углов здания

Блок угловой внутренний

Для формирования внутренних углов возводимого строения.

Блок поясной

Дополнительный строительный блок. Используется при устройстве армопояса, который делается по итогам каждого этажа для равномерного распределения нагрузки от перекрытий
